File Name: Server+DataRepeater.asp
Class Name: ServerDataRepeater

Overview
A data-bound WebControl that allows custom layout by repeating specified templates for each row in the DataSource.


Public Constructors
New_ServerDataRepeater(name) This constructor is used to create an instance of the control while at the same time assiging it a name and id.
New ServerDataRepeater This is the default constructor and it will create an instance of the control without a name. Controls instanciated this way must assign the name of the control manually by doing this: <controlVariable>.Control.Name = "<ControlName>"

Public Properties
Control WebControl Inherits attributes from the WebControl class.
DataSource ADODB.RecordSet Gets/Sets the datasource.
Header String Gets/Sets name of the function used to render the header.
ItemTemplate String Gets/Sets name of the function used to render a item.
AlternateItemTemplate String Gets/Sets name of the function used to render an alternating item. When specified, the repearter will alternate the calls to ItemTemplate and AlternatingItemTemplate
Footer String Gets/Sets name of the function used to render the footer.
SeparatorTemplate String Gets/Sets name of the function used to render the item separator.
