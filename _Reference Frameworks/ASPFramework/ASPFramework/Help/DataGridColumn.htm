DataGridColumn V1.1 F2.0

File Name: Server_DataGrid.asp
Class Name: DataGridColumn

Overview
This class is used to define the columns in a ServerDataGrid. They can be auto-generated by the ServerDataGrid or you can create your own array of DataGridColumns.


Public Constructors
New_DataGridColumn(ColumnType, DataTextField, DataValueField, Format) This constructor is used to create an instance of a DataGridColumn and provide some parameters at the same time.
New_DataGridTemplateColumn(CellRenderFunctionName) This constructor is used to create an instance of a DataGridColumn to be used as a TemplateColumn
New DataGridColumn This is the default constructor and it will create an instance of the control without a name. Controls instanciated this way must assign the name of the control manually by doing this: <controlVariable>.Control.Name = "<ControlName>"

Public Properties
ColumnType String '1 Bound, 2 Format, 3 Template, 4 CheckBox, 5 Radio
CellRenderFunctionName String Name of the function to be used to render the cell contents.
HorizonalAlign String Gets/Sets the of horizonal align property the cell.
VerticalAlign String Gets/Sets the of vertical align property of the cell.
ColumnWidth String Gets/Sets the column width.
DataTextField String Gets/Sets the name of the field used to render the cell contents..
DataValueField String Gets/Sets the name of the field used to render as a value for TemplateColumns, Radio or CheckBox.
HeaderText String Gets/Sets the column text header.
Visible Boolean Gets/Sets the a value to indicate wether or not the column should be rendered.
SortColumn String If <>"" then the column header will be displayed as a link and will raise an event when clicked. The event will have this format <control owner>.SortColumn(e)
NoWrap Boolean Gets/Sets the a value to indicate wether or not the column contents should be wrapped if they don't fit the width.
ReadyOnly Boolean True if the EditControl is Nothing
EditControl WebControl

WebControl used to edit the contents of the column when in EditMode. You must use a TextBox,DropDown,CheckBox,CheckBoxList or RadioButtonList (in any of the different modes).
DataFormatString String Used for ColumnType = 2. You can supply any string and the following tokens will be replaced: {0} with DataTextField and {1} with DataValueField. (i.e. "<a href='openuser.asp?id={1}'>{0}</a>"
DataSource

(Disconnected)

ADODB.RecordSet

 DataSource for the fields. Inherited form DataGridOwner
DataGridOwner Server_DataGrid Object that "owns" this DataGridColumn. Currently only the Server_DataGrid.
OrdinalPosition Int Ordinal position of the column (relative to/in the Columns array of the datagrid)
DataFormatFunctionName(value) String Name  of a function to be used to alter/format the contents of the data column.

RenderHeader() Renders the column as a header.
GetCheckedValue() Gets the value of the Radio Button if checked. Applies only to ColumnType = 5
