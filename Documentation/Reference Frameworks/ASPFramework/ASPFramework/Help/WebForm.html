CLASP - Classic ASP Framework.

CLASP Forms

A Clasp Form is basically an ASP page that makes use of the WebControl.asp and that includes the CLASP fields. To open the form you use Page.OpenForm and to close the form you use Page.CloseForm. To execute the  Page you use Page.Execute.

By doing this you immediately gain access to all server side events that CLASP offers. These events will help you you to structure like if you were working in ASP.NET or a Visual Basic Application.

A Clasp Page is organized in the following way:
  • Include WebControl.asp
  • Include server controls to be used
  • Include any extra library
  • Page.Execute
  • Page.OpenForm
  • HTML SECTION (With WebControls Embeeded)
  • Page.CloseForm
  • Control Declaration
  • Page Level variables
  • Page Events
  • PostBack Events
  • Helper Routines
